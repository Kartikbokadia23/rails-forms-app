# README

## rails forms app 

### Ruby version
ruby 2.7.2

### Rails version
rails 6.0.1

## Installation

1. Clone this repository using git clone.
```
git clone <My Repository Name>
```
2. Install required dependencies
```
bundle install
npm install
```
3. Setup the database
```
rails db:setup
```
4. migrate the database
```
rails db:migrate
```
5. Run the local server
```
rails s
```
